# Odia Voice-Based Government Scheme Assistant

## Author
Priyanka

## Overview
This project is a voice-first agentic AI system that helps users identify eligible government welfare schemes based on age, income, occupation, and category.

## Architecture
Planner → Executor → Evaluator with memory and failure handling.

## Setup
pip install -r requirements.txt

## Run
python main.py

## Note
Offline Windows TTS does not support Odia voices. Therefore, Odia text output is shown with an audible confirmation sound.
